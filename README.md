## Reminder App

A powerful, feature-rich reminder application built with UIKit and VIPER architecture, serving as a complete analogue to the native iOS Reminders app with enhanced functionality and Realm database persistence.

## 📱 Overview

A beautifully crafted reminder management app that helps users organize, track, and never miss their tasks. Built with clean VIPER architecture and Realm for robust local data persistence, providing a seamless native iOS experience.

## ✨ Features

Smart Lists - Create and manage multiple reminder lists (Personal, Work, Shopping, etc.)  
Rich Reminders - Add titles, notes, due dates, priorities, and tags  
Time Notifications - Set reminders with custom alert times  
Priority Levels - High, Medium, Low with color coding  
Search & Filter - Find reminders quickly with powerful search  
Quick Actions - Mark complete, snooze, or delete with swipe gestures  

## 🛠️ Tech Stack

UIKit - Native iOS UI framework  
VIPER - Clean architecture pattern  
Realm - Mobile database for local persistence  
UserNotifications - Local and push notifications
